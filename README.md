## Tech Career Shifters - Data and Analytics

This repository contains the outline to the weekly study group sessions of data and analytics subset of Tech Career Shifters community. The community aims to build an accountability group where everyone would support one another in their learning and career endeavors related to data and analytics (data analyst, data engineers, data scientist).

### SQL
